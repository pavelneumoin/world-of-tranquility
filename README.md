## 📋 О проекте

**«Мир спокойствия»** — это интерактивное веб-приложение, разработанное для помощи людям в управлении стрессом, тревожностью и паническими состояниями. Проект предоставляет комплекс научно обоснованных техник релаксации, дыхательных практик и инструментов для отслеживания эмоционального состояния.

### 🎯 Цель проекта

Создание доступного и удобного инструмента для:
- Снижения уровня стресса и тревожности
- Обучения эффективным техникам саморегуляции
- Отслеживания прогресса в практике релаксации
- Предоставления экстренной психологической поддержки

### 🌟 Актуальность

В современном мире проблема стресса и тревожности становится всё более острой. По данным ВОЗ, более 300 миллионов человек в мире страдают от тревожных расстройств. Наше приложение призвано помочь людям справляться с повседневным стрессом и улучшить качество жизни.

## ✨ Функциональность

### 🫁 Дыхательные практики (`/breathing`)

Страница с интерактивными дыхательными упражнениями:

- **Техника 4-7-8** — метод быстрого успокоения:
  - Вдох на 4 счёта
  - Задержка дыхания на 7 счётов
  - Выдох на 8 счётов
  
- **Квадратное дыхание** — техника для концентрации (4-4-4-4)

- **Диафрагмальное дыхание** — глубокое дыхание животом

Все упражнения сопровождаются анимированным визуальным таймером.

### 🧠 Техники релаксации (`/techniques`)

Подробные инструкции по научно обоснованным методам:

- **Прогрессивная мышечная релаксация** — поочерёдное напряжение и расслабление групп мышц
- **Техника заземления 5-4-3-2-1** — возвращение в настоящее через органы чувств
- **Визуализация безопасного места** — создание внутреннего ресурса спокойствия
- **Медитация осознанности** — наблюдение за мыслями без оценки

### 📊 Статистика (`/statistics`)

Отслеживание прогресса пользователя:

- Количество заполненных анкет
- Записи в личном журнале
- История измерений уровня стресса
- Анализ эффективности практик

### 📝 Анкетирование (`/questionnaire`)

Система оценки состояния до и после практик:

- Шкала уровня тревожности (0-10)
- Шкала уровня стресса (0-10)
- Оценка настроения
- Выбор наиболее эффективной техники

### 📖 Личный журнал (`/journal`)

Инструмент для рефлексии и отслеживания эмоционального состояния:

- Выбор текущего настроения (6 эмодзи-категорий)
- Свободное описание мыслей и ощущений
- История записей с датами
- Анализ паттернов настроения

### 🎵 Медиа-контент (`/media`)

Коллекция расслабляющего контента:

- **8 аудиодорожек для медитации:**
  - Дзен-медитация (поющие чаши)
  - Дождь в лесу
  - Морские волны
  - Тропический лес
  - Треск костра
  - Горный ручей
  - Белый шум
  - Медитативная музыка

- **Таймер медитации** с настройками на 5, 10, 15 минут
- **Видео природы** для визуальной релаксации

### 🆘 Экстренная помощь (`/emergency`)

Страница быстрой помощи при острых состояниях:

- **Телефоны доверия России:**
  - 8-800-2000-122 (детский)
  - 8-800-333-44-34 (кризисная линия)

- Интерактивное дыхательное упражнение
- Техника заземления 5-4-3-2-1
- План действий при панической атаке
- Критерии для обращения к специалисту

### 📈 Трекер стресса

На главной странице — интерактивный инструмент для быстрой оценки текущего уровня стресса по шкале от 1 до 10 с персонализированными рекомендациями.

## 🚀 Установка

### Локальный запуск

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/pavelneumoin/world-of-tranquility.git
cd world-of-tranquility
```

2. **Убедитесь, что установлен Python 3.8+:**
```bash
python --version
```

3. **Запустите сервер:**
```bash
python server.py
```

4. **Откройте в браузере:**
```
http://localhost:5000
```

### Развёртывание на Render

1. Подключите репозиторий к Render.com
2. Выберите тип сервиса: **Web Service**
3. Укажите команду запуска: `python server.py`
4. Приложение автоматически развернётся
